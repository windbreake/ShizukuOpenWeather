# CLI Design

Planned binary name: `weather`.

Commands:

```bash
weather search жІҲйҳі
weather current жІҲйҳі
weather summary жІҲйҳі
weather hourly жІҲйҳі --hours 24
weather daily жІҲйҳі --days 7
weather aqi жІҲйҳі
weather tui
weather serve
weather open жІҲйҳі
weather launch жІҲйҳі
```

CLI output should use card-like terminal formatting and support JSON output for scripts.
